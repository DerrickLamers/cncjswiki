## Key Features
* Supported CNC controller
  * Grbl v0.9 or later
  * TinyG2 will be supported in CNC.js v2.0
* 3D Visualizer
* Allows multiple HTTP connections at the same serial port
* Keyboard Shortcuts
* Contour ShuttleXpress support 
* Ready for i18n
* Widget management
  - Axes widget
  - Connection widget
  - Console widget
  - G-code widget
  - Grbl widget
  - Probe widget
  - Spindle widget
  - Webcam widget
* Z-Probing
